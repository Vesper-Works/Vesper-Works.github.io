Marching Cubes

Marching cubes is an algorithm for extracting a polyganol mesh from a three-dimentional scalar field.
It's used mainly for terrain generation. My implementation generation planetary terrain.
As of now I am working on making the algorithm into a Unity compute shader, allowing for real-time updating with no performance impact.

Boids

The boids algorithm simulations flocking behaviour in animals like birds and fish.
Using three simple rules:
Avoid other entities;
Match nearby entities' rotation;
Keep to the centre of the boid.

RTS Rimworld

RTS Rimworld is a personal project born out of my love for the game Rimworld.
The game takes Rimworld's building and combat system and pits players against eachother. I've developed a lot of back-end for the game, including a fully moddable tyd (xml varient) entity system.
This allows anyone to create anything in the game from terrain to new weapons and characters.
The game uses SmartFoxServer for networking.
