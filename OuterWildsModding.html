About Me


Outer Wilds Online

 Outer Wilds Online is an always-online MMO mod which connects players into the same solar system.
In order to get around synchronisation, all players are in their own universe, but projecting themselves and objects they interact with to other players, using various astrological objects as references, instead of world-space.
The project using the SDK SmartFoxServer for networking.

Outer Wilds Half-Life Overhaul

 The Half-Life Overhaul mod is a collaberative effort between myself and Xen, aiming to change all of the game's textures, sounds and character models with that of Half-Life One.

Outer Wilds Scout-Streaming
The Scout Streaming mod changes the player's scout camera into live footage.

Outer Wilds Shaderpack
The shader pack contains 5 different shaders. Each of these use Unity compute shaders, taking in the camera's render as input and outputting a modified view every frame.
